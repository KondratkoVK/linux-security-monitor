# linux-security-monitor
# 🛡️ Linux Security Monitoring Platform

⚠ **Проект находится в стадии активной разработки (Work in Progress).**
Функциональность системы постепенно расширяется и дополняется новыми модулями обнаружения атак и инструментами анализа безопасности.

---

# 📖 Описание проекта

Данный проект представляет собой платформу мониторинга безопасности Linux-сервера, предназначенную для обнаружения подозрительной активности, анализа системных логов и визуализации метрик в режиме реального времени.

Система реализует элементы **SOC (Security Operations Center)** для одного сервера и позволяет выявлять потенциальные атаки и аномалии.

Платформа сочетает инструменты мониторинга инфраструктуры и анализа безопасности.

---

# 🎯 Основные задачи системы

Система предназначена для:

* мониторинга состояния Linux-сервера
* анализа системных логов
* обнаружения атак и подозрительной активности
* визуализации метрик безопасности
* генерации предупреждений (alerts)

# 🔍 Реализованные механизмы обнаружения угроз

Система включает несколько модулей анализа безопасности.

## Обнаружение перебора SSH (Brute Force)

Анализируется файл:

```
/var/log/auth.log
```

Подсчитывается количество неудачных попыток входа.

Метрика:

```
ssh_failed_logins
```

При превышении порогового значения генерируется предупреждение.

---

## Обнаружение аномальной загрузки ресурсов

Позволяет выявлять:

* резкое увеличение загрузки CPU
* возможные fork-bomb атаки
* перегрузку системных ресурсов

---

## Обнаружение сетевого сканирования

Анализируются сетевые соединения для выявления:

* сканирования портов
* аномального количества соединений
* подозрительной сетевой активности

---

## Мониторинг системных событий (auditd)

Используется подсистема аудита Linux для отслеживания:

* изменений системных файлов
* подозрительных действий пользователей
* событий безопасности

---

## Анализ сетевого трафика

Используется анализ сетевых пакетов для обнаружения:

* аномального сетевого трафика
* подозрительных соединений
* сетевых атак

---

# 📊 Панель мониторинга

Система визуализации отображает:

* загрузку процессора
* использование оперативной памяти
* сетевую активность
* количество неудачных SSH-входов
* системную нагрузку
* события безопасности

Данные отображаются в режиме реального времени.

---

# 🚨 Система предупреждений

При обнаружении подозрительной активности система генерирует предупреждения.

Пример предупреждения:

```
ALERT: SSH Brute Force Attack
Host: Linux Server
Failed attempts: 12
Severity: Critical
```

# 🚀 Установка

Клонирование репозитория:

```
git clone https://github.com/yourusername/linux-security-monitor.git
cd linux-security-monitor
```

Запуск системы мониторинга:

```
docker compose up -d
```

# 🧪 Тестирование системы

Для тестирования обнаружения brute-force атаки выполните несколько неудачных попыток входа:

```
ssh user@localhost
```

После этого запустите скрипт обнаружения:

```
sudo ./detection/ssh_bruteforce.sh
```

Метрики будут доступны в системе мониторинга.
